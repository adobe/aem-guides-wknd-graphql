Web Component example for AEM headless

This is a static HTML web page that could be generated by any system, or framework, but includes a "person-info" web component. The "person-info" web component retrieves the "person data" from AEM via AEM's Headless GraphQL APIs, and renders the person using a custom web component implementation.

The "person-info" web component could be included on any web page, regardless of the framework or system that generated the web page. The web component provides a highly-portable mechanism for rendering AEM content in any web experience!
